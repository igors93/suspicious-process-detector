# Security Policy

## Purpose

Suspicious Process Detector is an educational and defensive security project.

It is designed to help users learn about process inspection, rule-based detection, and defensive security automation.

## Supported Versions

This project is currently in early development.

| Version | Supported |
|---|---|
| 0.1.x | Yes |

## Reporting Security Issues

If you find a security issue in this project, please open an issue with a clear description.

Do not include:

- Real malware samples
- Sensitive personal information
- Private system logs
- API keys, tokens, or credentials

## Responsible Use

This tool should only be used on systems you own or where you have explicit permission.

The project does not perform offensive actions, process termination, malware execution, exploitation, or unauthorized activity.
